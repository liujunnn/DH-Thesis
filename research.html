Research Methodology

A multi-stage, mixed-methods approach integrating computational data analysis, semantic web technologies, and critical theory to decolonize digital heritage

The Research Problem

Digital Orientalism

Contemporary digital infrastructures actively replicate the historical decontextualization of non-Western cultural artifacts. Western-centric metadata standards like CIDOC-CRM encode colonial perspectives, marginalizing other ways of knowing.

Meaning Stripping

The journey of Chinese porcelain from kilns to European collections involved systematic removal of original cultural context. Digital databases perpetuate this by forcing objects into hierarchical Western categories, creating "semantic decoupling."

Theoretical Foundation: Relational Accountability

Drawing from Indigenous Knowledge Organization Systems (IKOS), this research applies the principle of "relational accountability" - where an object's meaning derives from its web of relationships rather than isolated physical properties.

Research Methodology

Data Collection from Europeana

Programmatically extracted metadata for Chinese porcelain using the Europeana Search API, yielding 3,372 records from 107 European institutions. 

query = '"Chinese porcelain"'
records = query_europeana_api(query)
# Result: 3,372 JSON objects with metadata fields

Data Cleaning and Preprocessing

Harmonized inconsistent metadata using Python Pandas: flattened JSON structures, standardized dating formats, cleaned textual data, and handled missing values.

Key Challenges
  • • 72.7% missing kiln attribution
  • • Multiple dating formats
  • • Multilingual descriptions
Solutions Applied
  • • Regex pattern matching
  • • Date standardization
  • • Text normalization

Relational Ontology Development

Created a non-hierarchical ontological framework using Protégé, designed to hold Chinese production knowledge and European reception history in dialogue.

Context of Production

  • • Kiln sites (Jingdezhen, Dehua)
  • • Production techniques
  • • Dynasty/reign periods
  • • Original symbolic meanings

Context of Reception

  • • Trade routes (VOC)
  • • European provenance
  • • Collecting patterns
  • • Cultural reinterpretation
